# Code of Conduct

KnowledgeOps Agent aims to be a practical, respectful, and technically rigorous open source project.

## Expected Behavior

- Be respectful and constructive in issues, pull requests, and discussions.
- Focus feedback on the work, tradeoffs, and evidence.
- Assume good intent while still asking for clarity when something is ambiguous.
- Help keep conversations useful for future readers.

## Unacceptable Behavior

- Harassment, insults, threats, or discriminatory language.
- Publishing private information without permission.
- Repeated off-topic or disruptive comments.
- Security disclosure details posted publicly before a fix is available.

## Enforcement

Maintainers may edit, hide, or remove comments and may restrict participation for behavior that harms the project community.

For security-sensitive reports, follow [SECURITY.md](SECURITY.md).
